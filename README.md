# Tanu Portfolio

Animated portfolio website built with:

- Next.js (App Router + TypeScript)
- Tailwind CSS
- GSAP (+ ScrollTrigger)

## Run locally

```bash
npm install
npm run dev
```

## Scalable structure

```text
src/
  app/                    # routes, layout, global styles
  components/
    sections/             # feature/section-level UI
    ui/                   # reusable shared UI pieces
  data/                   # resume-driven content objects
  lib/                    # gsap setup and utilities
```
